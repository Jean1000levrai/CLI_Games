### 1. Prerequisites
 - C99 or above
 - A Compiler (e.g. gcc, glang, ...)

### 2. Download

#### In the desired folder:

```
git clone git@github.com:Jean1000levrai/CLI_Games.git
cd CLI_Games
```

```
cd ticTacToe    # for example

```


### 4. Play on the command line interface!
```
./main
```
