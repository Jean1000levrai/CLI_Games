## ðŸ’¾ Installation
### 1. Prerequisites
 - C99 or above
 - A Compiler (e.g. gcc, clang, ...)
 - CMake >= 3.16     

### 2. Download

#### In the desired folder:

```
git clone https://github.com/Jean1000levrai/CLI_Games.git
cd CLI_Games
```
#### Browse to the game of your choice
```
cd ticTacToe    # for example
```

### 3. Compile
```
mkdir -p build
cd build/
cmake ..
make
```

### 4. Play on the command line interface!
```
./main
```
